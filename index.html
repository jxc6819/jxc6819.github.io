Submarine Combat System

This project implements a comprehensive submarine combat system with realistic physics simulation. The game features advanced torpedo tracking algorithms, depth-based pressure mechanics that affect submarine maneuverability, and a sonar detection system. Players must manage their submarine's buoyancy, speed, and stealth capabilities while engaging enemy vessels in tactical underwater warfare.

Key technical achievements include implementing custom physics for underwater movement, developing an AI system for enemy submarines that uses predictive targeting, and creating a dynamic audio system that simulates sound propagation through water.
